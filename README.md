# Phoenix Page and Mind

# Overview
Phoenix Page and Mind is a business application built in **VB.NET**, designed for both coursework and real-world use.  
It integrates modular coding, clear interface wiring, and practical project management concepts to demonstrate professional standards in software development.

# Features
- Modular VB.NET code structure with parameters and lists
- User-friendly interface wired to backend logic
- Resource management (forms, modules, and `.resx` files)
- Professional documentation and reporting standards
- Scalable design for coursework and entrepreneurial projects
